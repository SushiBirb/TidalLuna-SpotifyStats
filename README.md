# TidalLuna-SpotifyStats

A plugin for Tidal Luna that replaces the native Tidal "Fans" counts with exact Spotify Monthly Listeners, and appends Spotify play counts next to tracks on album pages!

### Disclaimer: Vibecoding
This plugin was entirely "vibecoded" by AI. That means it was written by an LLM assistant who just went with the flow and scraped Spotify through DuckDuckGo to get you the listener numbers without needing API keys. Because it relies on scraping search engine snippets, it might occasionally misbehave, pick up the wrong artist if they share a name, or break if Spotify/DuckDuckGo changes their HTML structure. Use at your own vibe!

### Installation
1. Open TidaLuna settings.
2. Go to the Plugins tab.
3. Paste the URL `https://github.com/SushiBirb/TidalLuna-SpotifyStats` into the "Install from GitHub" box.
4. Restart Tidal.
