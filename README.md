# TidalLuna-SpotifyStats
A plugin for TidaLuna that injects Spotify Listeners and Last.fm Playcounts directly into Tidal's artist and album pages.

## Disclaimer
This code was created with the help of AI (vibecoding). While it aims to improve your Tidal experience with cross-platform metadata, it may break if Tidal updates its layout or if API restrictions change. Use at your own discretion!

## Installation
In TidaLuna, go to the Plugins menu, paste the URL of this repository, and click Add.
URL: `https://raw.githubusercontent.com/SushiBirb/TidalLuna-SpotifyStats/main/package`
*(Make sure to append `/package` so it downloads the correct `package.json` file)*
